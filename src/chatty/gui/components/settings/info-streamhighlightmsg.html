The success message that gets output when adding a Stream Highlight (error messages can not be customized). The following parameters can be used:

  • $(added) - "highlight" or "highlight/marker" depending on settings
  • $(chan) - the channel name
  • $(uptime) - the stream uptime
  • $(comment) - shortened comment in parentheses
  • $(fullcomment) - the entire comment in parentheses
  • $(chatuser) - the user that executed the chat command, or empty if triggered locally

Example:

Added stream highlight [$(uptime)] $(comment) $if(chatuser,\(This is a mod-only command\))
Compared to default this omits some information, but adds a note about who can run the chat command (but only if actually ran from chat command, note escaping the parentheses as well).
